<h1>Cryptocurrency Telegram Bot</h1>

<p>
A Telegram bot for tracking cryptocurrency prices, managing a portfolio,
setting price alerts, and converting crypto assets.
</p>

<h2>Features</h2>

<h3>Prices & Market Data</h3>
<ul>
  <li>View prices of popular cryptocurrencies</li>
  <li>Get detailed price information for a specific coin</li>
  <li>Top cryptocurrencies by market capitalization</li>
  <li>Historical price data (7, 30, 90 days)</li>
  <li>Global crypto market statistics</li>
  <li>Fear & Greed Index</li>
</ul>

<h3>Exchange & Conversion</h3>
<ul>
  <li>Cryptocurrency conversion calculator</li>
  <li>Supports any available crypto asset</li>
</ul>

<h3>Search</h3>
<ul>
  <li>Search cryptocurrencies by name or symbol</li>
</ul>

<h3>Portfolio</h3>
<ul>
  <li>Track your crypto investments</li>
  <li>Add and remove assets</li>
  <li>Automatic portfolio value calculation</li>
</ul>

<h3>Favorites</h3>
<ul>
  <li>Quick access to favorite cryptocurrencies</li>
  <li>Track selected assets</li>
</ul>

<h3>Alerts</h3>
<ul>
  <li>Price alerts (above or below a target price)</li>
  <li>Automatic price checks every minute</li>
</ul>

<h3>Balance & Payments</h3>
<ul>
  <li>Balance management</li>
  <li>Deposit via CLICK Uzbekistan</li>
  <li>Telegram Payments support</li>
</ul>

<h2>Getting Started</h2>

<p>Clone the repository:</p>
<pre><code>git clone https://github.com/it-withend/Simple-CryptoTracker-bot.git
cd Simple-CryptoTracker-bot</code></pre>

<p>Install dependencies:</p>
<pre><code>pip install -r requirements.txt</code></pre>

<p>Create and configure environment variables:</p>
<pre><code>cp env.example .env</code></pre>

<pre><code>TELEGRAM_BOT_TOKEN=your_telegram_bot_token
CLICK_PROVIDER_TOKEN=your_click_provider_token</code></pre>

<p>Run the bot:</p>
<pre><code>python crypto_bot.py</code></pre>


<h2>Supported Cryptocurrencies</h2>
<p>
The bot supports all cryptocurrencies available on CoinGecko (10,000+ assets).
</p>
